# MuJoCo 物理仿真从零开始学习计划

本学习计划专为零基础学习者设计，旨在通过“原理讲解 -> API/XML语法说明 -> 代码实战 -> 课后练习”的闭环模式，帮助你系统掌握 MuJoCo 仿真的核心技能。

> [!NOTE]
> **关于版本选择**：
> 虽然当前工作区目录名为 `mujoco-py`（旧版 Python 绑定，配置繁琐且已停止维护），本教程将全面基于 DeepMind 官方维护的**最新版官方 Python 绑定 (`import mujoco`)**进行讲解。这是目前的行业标准，安装简单，API 也更直观。

## 课程大纲

### 第一章：MuJoCo 基础与环境搭建
*   **学习目标**：理解 MuJoCo 的核心架构，成功安装并运行第一个仿真窗口。
*   **理论要点**：
    *   MuJoCo 的核心组件：`mjModel`（静态物理描述）与 `mjData`（动态运行状态）。
    *   仿真的时间步长（`timestep`）与数值积分。
*   **实战内容**：
    *   搭建 Python 虚拟环境并安装官方 `mujoco` 库。
    *   编写第一个 Hello World 脚本，加载极简 XML 并拉起仿真渲染器。
*   **代码输出**：`ch1_hello_mujoco.py`

### 第二章：MJCF 建模语言与模型构建
*   **学习目标**：掌握 MJCF（MuJoCo XML）语法，能够自己动手建立机器人/刚体物理模型。
*   **理论要点**：
    *   MJCF 的层级结构：`<worldbody>`（世界坐标系）、`<bodybody>`（刚体）、`<geom>`（碰撞/外观几何体）与 `<joint>`（关节）。
    *   4种关节类型详解：`hinge`（转动关节）、`slide`（滑动关节）、`ball`（球关节）、`free`（自由关节）。
    *   惯性参数（`inertial`）与材质/光源的配置。
*   **实战内容**：
    *   从零编写一个双摆（Double Pendulum）的 XML 配置文件。
    *   为双摆配置不同的关节阻尼（`damping`）和摩擦（`friction`）。
*   **代码输出**：`ch2_double_pendulum.xml`，`ch2_load_pendulum.py`

### 第三章：Python API 基础与状态获取
*   **学习目标**：深入理解 Python 仿真控制循环，学会实时读取物体的状态数据（位置、速度、加速度等）。
*   **理论要点**：
    *   仿真核心循环机制：`mujoco.mj_step(model, data)`。
    *   物理量在 `mjData` 中的存储方式：
        *   `qpos`（广义位置，注意自由关节的四元数表示）
        *   `qvel`（广义速度）
        *   `qacc`（广义加速度）
        *   `xpos`（笛卡尔空间中物体的绝对位置）
*   **实战内容**：
    *   编写一个小球自由落体的仿真。
    *   实时打印并使用 `matplotlib` 绘制小球的高度随时间变化的曲线。
*   **代码输出**：`ch3_free_drop.py`

### 第四章：执行器与关节控制 (Actuators & Control)
*   **学习目标**：学会让模型“动起来”，掌握如何给关节施加驱动力，以及如何实现基础的控制算法（如 PID）。
*   **理论要点**：
    *   MuJoCo 中的执行器类型：`motor`（直接力矩驱动）、`position`（位置伺服）、`velocity`（速度伺服）。
    *   控制变量的注入方法：`mjData.ctrl`。
    *   经典控制算法：PD 控制（比例-微分控制）在仿真中的实现。
*   **实战内容**：
    *   在双摆模型的关节上配置力矩执行器（motor）。
    *   编写 Python 控制算法，使双摆的目标关节稳定维持在指定的角度（PD控制）。
*   **代码输出**：`ch4_pendulum_control.py`

### 第五章：传感器、碰撞与接触力 (Sensors & Contacts)
*   **学习目标**：学会获取仿真环境的反馈信息，理解物体间的碰撞与接触力学。
*   **理论要点**：
    *   在 XML 中配置 `<sensor>` 标签：力矩传感器、触觉传感器、IMU（加速度计/陀螺仪）。
    *   碰撞的判定规则：Geom 组（group）与碰撞矩阵（condim）。
    *   接触动力学：摩擦圆锥（friction cone）与接触约束（contact constraints）。
*   **实战内容**：
    *   搭建一个小球撞击地面的仿真。
    *   读取传感器数据，并利用 Python 获取小球与地面在撞击瞬间的法向接触力。
*   **代码输出**：`ch5_contact_sensor.py`

### 第六章：动态控制回调与重力补偿 (Dynamic Callbacks)
*   **学习目标**：掌握 MuJoCo 的底层控制回调函数机制，实现经典动力学控制方法（如重力补偿）。
*   **理论要点**：
    *   底层控制周期与控制回调 `mujoco.set_mjcb_control`。
    *   多自由度刚体动力学中的重力项计算与补偿原理。
*   **实战内容**：
    *   在双摆模型中，利用物理引擎自带的动力学计算，在每步仿真前注入重力补偿力矩，实现“重力消失”的仿真效果。
*   **代码输出**：`ch6_gravity_compensation.py`

### 第七章：Gymnasium 强化学习自定义环境构建 (Custom RL Env)
*   **学习目标**：学会将 MuJoCo 仿真模型包装为符合 OpenAI/Farama Gymnasium 标准的强化学习环境，这是具身智能的基础。
*   **理论要点**：
    *   马尔可夫决策过程 (MDP) 在机器人仿真中的映射。
    *   状态空间 (Observation Space) 与动作空间 (Action Space) 的定义。
    *   奖励函数设计原则：任务引导奖励 (Task reward) 与正则项（Control cost / Joint velocity penalty）。
    *   环境重置 (Reset) 机制中的随机初始化。
*   **实战内容**：
    *   继承 `gymnasium.Env`（或 `gymnasium.utils.EzPickle`），将双摆模型改写为标准的强化学习环境。
*   **代码输出**：`ch7_gym_env.py`

### 第八章：强化学习算法训练与策略回放 (RL Training)
*   **学习目标**：使用主流强化学习库（Stable-Baselines3）训练智能体，并回放训练好的策略。
*   **理论要点**：
    *   无模型强化学习算法简介（PPO、SAC）。
    *   训练状态的监控（TensorBoard）与超参数调节。
*   **实战内容**：
    *   使用 SB3 的 PPO 算法在自定义的双摆环境上进行训练。
    *   记录训练日志并在 TensorBoard 中查看收敛曲线。
    *   加载训练好的模型，在 MuJoCo 3D 渲染窗口中展示策略效果。
*   **代码输出**：`ch8_train_rl.py`

### 第九章：具身智能基础：相机传感器与视觉强化学习 (Vision-based RL)
*   **学习目标**：掌握在 MuJoCo 中配置相机并读取 RGB-D（彩色及深度）图像数据，实现基于视觉的控制（Eye-in-Hand / Eye-to-Hand）。
*   **理论要点**：
    *   MJCF 中的相机标签 `<camera>` 配置与内外参理解。
    *   离屏渲染 (Off-screen rendering) 技术。
    *   视觉观测空间设计与深度图数据处理。
*   **实战内容**：
    *   在环境中添加相机，利用 Python 读取仿真图像并使用 `PIL` 或 `OpenCV` 可视化。
    *   构建一个以图像/深度图为输入的简易 RL 观测器。
*   **代码输出**：`ch9_camera_rl.py`

### 第十章：具身智能实战：机械臂控制与任务仿真 (Robotic Manipulation)
*   **学习目标**：在 MuJoCo 中加载多自由度机器人，结合逆运动学 (IK) 与强化学习实现目标到达或抓取任务。
*   **理论要点**：
    *   关节空间控制 (Joint Space Control) 与笛卡尔空间控制 (Operational Space Control)。
    *   逆运动学 (Inverse Kinematics) 原理。
    *   抓取/到达任务的奖励函数设计（包含距离惩罚、接触判定）。
*   **实战内容**：
    *   导入一个简易机械臂（如 3 自由度机械臂或 Panda 机械臂）。
    *   设计一个“随机目标点到达 (Reaching Task)”任务，并使用 PPO 训练机械臂的控制策略。
*   **代码输出**：`ch10_robot_reach.py`

## 学习规范与方法
1.  **逐章推进**：我们将采用交互式学习。你确认大纲无误后，我将为你编写**第一章**的教学文档与配套脚本，并指导你在本地运行。
2.  **代码注释规范**：所有的 Python 脚本首行均会注明该脚本的**学习目标**，并在关键步骤配以详尽的中文注释。
3.  **运行与反馈**：每一章的代码，建议你在本地运行并观察渲染效果。遇到报错或无法理解的物理现象，随时在对话中贴出，我们共同 Debug 和探讨。
